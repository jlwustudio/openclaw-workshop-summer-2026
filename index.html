💡 上機操作小提醒

工作目錄（workspace）：OpenClaw 的 Agent 預設以它自己的 workspace 資料夾（~/.openclaw/workspace）作為工作目錄，與你在終端機的哪個位置啟動無關。開始 Day 2 實作前，請先把 openclaw-sample-data 複製進 workspace（安裝教學 Part C 有完整步驟），Agent 才能用相對路徑直接讀到範例素材。想先看每個主題配哪個檔、該對 Agent 說什麼，可查閱範例檔案總覽。

安全防護：OpenClaw 會讀寫你電腦中的檔案並執行指令，在執行刪除、發送訊息或可能不可逆的動作時，記得請 Agent 「先講計畫，經過我同意後再動手」。你擁有最高指揮權，隨時可以切斷執行。
