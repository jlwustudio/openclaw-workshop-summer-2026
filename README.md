# 🎓 AI 召喚師：從使用 AI，到指揮 AI 工作坊

歡迎來到「AI 召喚師」實作工作坊！這是一個專為高中生設計的兩天探索旅程。在這裡，你將不再只是與 AI 一問一答聊天，而是學會如何給它「手」與「腳」，指揮它在你的電腦上自動上網、分類檔案、甚至接到你的手機 Telegram 裡！

🌐 **工作坊線上網頁**：[https://jlwustudio.github.io/openclaw-workshop-summer-2026/](https://jlwustudio.github.io/openclaw-workshop-summer-2026/)
> ⚠️ **注意**：線上網頁僅供課堂期間快速存取，**課程結束後該網站將會關閉**。屆時若想複習或閱讀教材，請直接點擊本機下載包中的 `index.html` 開啟即可。

## 📂 這個資料夾裡有什麼？

- **`index.html`**：**我們的學習傳送門（首頁）**。在你的電腦上直接點兩下打開它，就能輕鬆跳轉到所有的講義與教學。
- **`openclaw-day1-concepts-slides.pdf`**：**觀念大腦：AI 召喚術投影片 (PDF)**。第一天上午的概念講解簡報，建立大型語言模型（LLM）、AI Agent 與 Agentic AI 的基本認知，結尾附隨堂觀念挑戰小測驗。
- **`openclaw-install-guide.html`**：**裝備整備：環境建置與配置指南**。帶你一步步體檢電腦、啟動本機助理。
- **`openclaw-day1-afternoon.html`**：**新手實彈：Day 1 基礎實作手冊**。第一天下午的練習，教你如何下對指令、當個好的 AI 教練。
- **`openclaw-day2-worksheets.html`**：**高階召喚：Day 2 進階應用實作手冊**。第二天的主題任務，包含網頁自動化、檔案整理、Telegram 串接等。
- **`openclaw-sample-data/`**：**實作練習素材**。裡面放了我們這兩天要給 AI 助理練習的「玩具」，例如：
  - 待分類的混亂相簿與文件
  - 學生社團滿意度 CSV 調查表
  - 自製 Quiz 抽考機的學科題庫
  - 單字表與名言佳句

## 💻 怎麼開始使用？

1. **下載與解壓縮專案**：
   - 請在 GitHub 頁面點擊右上方綠色的 **Code** 按鈕，接著選擇 **Download ZIP** 下載專案壓縮檔。
   - 下載完成後，將該 ZIP 壓縮檔解壓縮至你的本機電腦（例如桌面）。
2. **開啟首頁**：
   - 進入解壓縮後的專案資料夾，點兩下打開 **`index.html`**，即可進入工作坊教材導覽首頁！

## 🎒 Day 2 前準備好

- **手機先裝好並登入 Telegram**：Day 2 有一個主題會把 AI 助理接到你的手機 Telegram 上。**Telegram 註冊需要用你的手機門號收簡訊驗證**，請務必**在來上課之前**就先裝好 App、完成帳號登入。若等到課堂才註冊，會卡在「等簡訊」而做不到真正好玩的部分。（真的沒有 Telegram 也沒關係，屆時可以和同學共用一支手機。）

## 🔒 安全守則

AI 助理擁有讀寫你電腦檔案與執行指令的權力。在操作時請務必記住：
1. **最高指揮權在你手上**：隨時可以在終端機切斷它的執行。
2. **請它先講計畫**：在涉及刪除檔案、送出訊息時，記得對它說：「先告訴我你的計畫，等我同意你再動手」。
